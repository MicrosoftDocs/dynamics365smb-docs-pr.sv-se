<span data-ttu-id="29ed8-104">Du kan importera och exportera en befintlig anpassad rapportlayout som en fil till och från en plats på datorn och nätverket.</span><span class="sxs-lookup"><span data-stu-id="29ed8-104">You can import and export an existing custom report layout as a file to and from a location on your computer and network.</span></span> <span data-ttu-id="29ed8-105">Du kan till exempel exportera en rapportlayout och sedan skicka filen till en annan person för att ändra.</span><span class="sxs-lookup"><span data-stu-id="29ed8-105">For example, you can export a report layout, and then send the file to another person to modify.</span></span> <span data-ttu-id="29ed8-106">Den personen kan sedan göra ändringar i layouten och returnera filen till dig så att du kan importera tillbaka den till.</span><span class="sxs-lookup"><span data-stu-id="29ed8-106">That person can then make the modifications to layout and return the file to you so that you can import it back.</span></span>

### <a name="to-export-a-report-layout-to-a-file"></a><span data-ttu-id="29ed8-108">Så här exporterar du en rapportlayout till en fil</span><span class="sxs-lookup"><span data-stu-id="29ed8-108">To export a report layout to a file</span></span>  
  
1.  <span data-ttu-id="29ed8-109">Välj ikonen ![Söka efter sida eller rapport](media/ui-search/search_small.png "Ikonen Söka efter sida eller rapport"), ange **Val av rapportlayout** och välj sedan relaterad länk.</span><span class="sxs-lookup"><span data-stu-id="29ed8-109">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
2.  <span data-ttu-id="29ed8-110">Markera raden för rapporten som innehåller den anpassade layout som du vill exportera, och sedan på fliken **Start**, i gruppen **Process** , välj **Anpassa layouter**.</span><span class="sxs-lookup"><span data-stu-id="29ed8-110">Select the row for the report that contains the custom report layout that you want to export, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
3.  <span data-ttu-id="29ed8-111">I fönstret **Rapportlayouter** markerar du den rapportlayout som du vill exportera till en fil, och sedan på fliken **Start** i gruppen **Process** och väljer **Exportera layout**.</span><span class="sxs-lookup"><span data-stu-id="29ed8-111">In the **Report Layouts** window, select the report layout that you want to export to a file, and then on the **Home** tab, in the **Process** group, choose **Export Layout**.</span></span>  
  
4.  <span data-ttu-id="29ed8-112">Välj **Spara** i dialogrutan **Exportera fil** och spara sedan filen på ett ställe på datorn eller nätverket.</span><span class="sxs-lookup"><span data-stu-id="29ed8-112">In the **Export File** dialog box, choose **Save**, and then save the file to a location on your computer or network.</span></span>  
  
### <a name="to-import-a-report-layout-file"></a><span data-ttu-id="29ed8-113">Så här importerar du en rapportlayoutfil</span><span class="sxs-lookup"><span data-stu-id="29ed8-113">To import a report layout file</span></span>  
  
1.  <span data-ttu-id="29ed8-114">Se till att den relevanta filen som definierar rapportlayouten är tillgänglig på din dator eller i ditt nätverk.</span><span class="sxs-lookup"><span data-stu-id="29ed8-114">Make sure that the relevant file that defines the report layout is available on your computer or network.</span></span>  
  
     <span data-ttu-id="29ed8-115">En Word-rapportlayoutfil måste ha filtypstillägget .docx.</span><span class="sxs-lookup"><span data-stu-id="29ed8-115">A Word report layout file must have the .docx file type extension.</span></span> <span data-ttu-id="29ed8-116">En RDLC-rapportlayoutfil måste ha filtypstillägget .rdlc eller .rdl.</span><span class="sxs-lookup"><span data-stu-id="29ed8-116">An RDLC report layout file must have the .rdlc or .rdl file type extension.</span></span>  
  
2.  <span data-ttu-id="29ed8-117">Välj ikonen ![Söka efter sida eller rapport](media/ui-search/search_small.png "Ikonen Söka efter sida eller rapport"), ange **Val av rapportlayout** och välj sedan relaterad länk.</span><span class="sxs-lookup"><span data-stu-id="29ed8-117">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Report Layout Selection**, and then choose the related link.</span></span>  
  
3.  <span data-ttu-id="29ed8-118">Markera raden för rapporten som du vill importera rapportlayouten till och sedan på fliken **Start**, i gruppen **Process** , välj **Anpassa layouter**.</span><span class="sxs-lookup"><span data-stu-id="29ed8-118">Select the row for the report to which you want to import the report layout, and then on the **Home** tab, in the **Process** group, choose **Custom Layouts**.</span></span>  
  
4.  <span data-ttu-id="29ed8-119">I fönstret **Rapportlayouter** markerar du den rapportlayout som du vill importera en fil till, och sedan på fliken **Start** i gruppen **Process** och väljer **Importera layout**.</span><span class="sxs-lookup"><span data-stu-id="29ed8-119">In the **Report Layouts** window, select the report layout to which you want to import the file, and then on the **Home** tab, in the **Process** group, choose **Import Layout**.</span></span>  
  
5.  <span data-ttu-id="29ed8-120">Välj dokumentet som definierar rapportlayouten i dialogrutan **Importera** och välj sedan **Öppna**.</span><span class="sxs-lookup"><span data-stu-id="29ed8-120">In the **Import** dialog box, select the document that defines the report layout, and then choose **Open**.</span></span>
