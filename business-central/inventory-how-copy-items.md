# <a name="copy-existing-items-to-create-new-items"></a><span data-ttu-id="103ed-103">Kopiera befintliga artiklar till Skapa nya artiklar</span><span class="sxs-lookup"><span data-stu-id="103ed-103">Copy Existing Items to Create New Items</span></span>
<span data-ttu-id="103ed-104">När du lägger till ett nytt objekt, för att spara tid, kan du använda funktionen **Kopiera objekt** för att kopiera ett befintligt objekt så att det används som mall för ett nytt objekt.</span><span class="sxs-lookup"><span data-stu-id="103ed-104">When you add a new item, to save time, you can use the **Copy Item** function to copy an existing item to use as a template for a new item.</span></span>  

## <a name="to-copy-an-existing-item-to-a-new-item"></a><span data-ttu-id="103ed-105">Kopiera ett befintligt objekt till ett nytt objekt.</span><span class="sxs-lookup"><span data-stu-id="103ed-105">To copy an existing item to a new item</span></span>  
1. <span data-ttu-id="103ed-106">Välj ikonen ![Glödlampa som öppnar funktionen Berätta](media/ui-search/search_small.png "Glödlampa som öppnar funktionen Berätta") och ange **Artiklar** och välj sedan relaterad länk.</span><span class="sxs-lookup"><span data-stu-id="103ed-106">Choose the ![Lightbulb that opens the Tell Me feature](media/ui-search/search_small.png "Tell me what you want to do") icon, enter **Items**, and then choose the related link.</span></span>  
2. <span data-ttu-id="103ed-107">Välj objektet som du vill kopiera och välj sedan åtgärden **Kopiera objekt**.</span><span class="sxs-lookup"><span data-stu-id="103ed-107">Select the item that you want to copy, and then choose the **Copy Item** action.</span></span>  
3. <span data-ttu-id="103ed-108">På sidan **Kopiera objekt** fyller du i fälten.</span><span class="sxs-lookup"><span data-stu-id="103ed-108">On the **Copy Item** page, fill in the fields.</span></span> [!INCLUDE[tooltip-inline-tip](includes/tooltip-inline-tip_md.md)]

    > [!NOTE]  
    > <span data-ttu-id="103ed-109">Du kan ange ett visst artikel nummer i fältet **Målartikelns nummer** som du vill hämta den nya artikeln till.</span><span class="sxs-lookup"><span data-stu-id="103ed-109">You can enter a specific item number in the **Target Item No.** field that you want the new item to get.</span></span> <span data-ttu-id="103ed-110">Alternativt kan du välja en befintlig nummerserie i fältet **Nummerserie för mål**.</span><span class="sxs-lookup"><span data-stu-id="103ed-110">Alternatively, you can select an existing number series in the **Target No. Series** field.</span></span> <span data-ttu-id="103ed-111">Den nya artikeln tilldelas då nästa lediga nummer i nummerserien.</span><span class="sxs-lookup"><span data-stu-id="103ed-111">The new item will then be assigned the next available number in that number series.</span></span>  

5. <span data-ttu-id="103ed-112">Välj **OK** om du vill kopiera dina val till det nya objektet.</span><span class="sxs-lookup"><span data-stu-id="103ed-112">To copy your selections to the new item, choose the **OK** button.</span></span>
