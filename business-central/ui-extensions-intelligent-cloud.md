# <a name="intelligent-cloud-base-extension"></a>Tillägget intelligent moln-bas

Det här tillägget tar igenom processen att konfigurera och hantera din intelligent moln-miljö. När din intelligent moln-miljö har konfigurerats kan du kan hantera datareplikeringar från din on-premises-lösning på din [!INCLUDE[prodshort](includes/prodshort.md)] i molnet. Detta tillägg fungerar tillsammans med ett tillägg för datareplikering som finns för din specifika lokala lösning. Mer information finns i [Intelligenta insikter](about-intelligent-cloud.md).
