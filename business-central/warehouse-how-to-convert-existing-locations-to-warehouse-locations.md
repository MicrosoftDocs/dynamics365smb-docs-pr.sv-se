    <span data-ttu-id="7719f-116">Med hjälp av öppna artikeltransaktioner för angivet lagerställe skapas dist.lager journalrader som summerar alla kombinationer av artikelnummer, variantkod, enhetskod och, vid behov, partinr och serienummer kombineras i artikeltransaktionerna.</span><span class="sxs-lookup"><span data-stu-id="7719f-116">Using the open item ledger entries for the specified location, warehouse journal lines are created that sum up every combination of Item No., Variant Code, Unit of Measure Code, and, if necessary, Lot No. and Serial No. in the item ledger entries.</span></span> <span data-ttu-id="7719f-117">Distributionslagerjorunalraderna bokförs sedan.</span><span class="sxs-lookup"><span data-stu-id="7719f-117">The warehouse journal lines are then posted.</span></span> <span data-ttu-id="7719f-118">Vid bokföringen skapas distributionslagertransaktioner som placerar lagret i distributionslagrets justeringslagerplats.</span><span class="sxs-lookup"><span data-stu-id="7719f-118">This posting creates warehouse entries that place the inventory in the warehouse adjustment bin.</span></span> <span data-ttu-id="7719f-119">**Justering lagerplatskod** anges också på lagerställekortet.</span><span class="sxs-lookup"><span data-stu-id="7719f-119">The **Adjustment Bin Code** on the location card is also set.</span></span>  

4.  <span data-ttu-id="7719f-120">Om du vill se vilka artiklar som lagts till i justeringslagerplatsen under batch-jobbet kan du köra rapporten **Dist.lager just.lagerplats**.</span><span class="sxs-lookup"><span data-stu-id="7719f-120">To see which items were added to the adjustment bin during the batch job, run the **Warehouse Adjustment Bin** report.</span></span>

> [!NOTE]  
>  <span data-ttu-id="7719f-123">Du bör köra det här batch-jobbet **Skapa dist.lagerplats** vid ett tillfälle då det inte påverkar det dagliga arbetet i systemet.</span><span class="sxs-lookup"><span data-stu-id="7719f-123">It is recommended that you run the **Create Warehouse Location** batch job at a time when it will not impact the daily work in the system.</span></span> <span data-ttu-id="7719f-124">I det här jobbet bearbetas alla transaktioner i tabellen **Artikeltransaktion** och om det finns många artikeltransaktioner kan jobbet ta flera timmar.</span><span class="sxs-lookup"><span data-stu-id="7719f-124">This job processes each entry in the **Item Ledger Entry** table, and if there are a large number of item ledger entries, the job can last several hours.</span></span>
