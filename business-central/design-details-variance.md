# <a name="design-details-variance"></a><span data-ttu-id="404f1-103">Designdetaljer: Varians</span><span class="sxs-lookup"><span data-stu-id="404f1-103">Design Details: Variance</span></span>
<span data-ttu-id="404f1-104">Varians definieras som skillnaden mellan den faktiska kostnaden och standardkostnaden, enligt beskrivningen i följande formel.</span><span class="sxs-lookup"><span data-stu-id="404f1-104">Variance is defined as the difference between the actual cost and the standard cost, as described in the following formula.</span></span>  

 <span data-ttu-id="404f1-105">faktisk kostnad – standardkostnad = varians</span><span class="sxs-lookup"><span data-stu-id="404f1-105">actual cost – standard cost = variance</span></span>  

 <span data-ttu-id="404f1-106">Om den faktiska kostnaden ändra, till exempel eftersom du bokför en artikelomkostnad på ett senare datum, kommer variansen att uppdateras därefter.</span><span class="sxs-lookup"><span data-stu-id="404f1-106">If the actual cost changes, for example, because you post an item charge on a later date, then the variance is updated accordingly.</span></span>  

> [!NOTE]  
>  <span data-ttu-id="404f1-107">Omvärderingen påverkar inte avvikelseberäkningen, eftersom omvärderingen endast ändrar lagervärdet.</span><span class="sxs-lookup"><span data-stu-id="404f1-107">Revaluation does not affect the variance calculation, because revaluation only changes the inventory value.</span></span>
