<span data-ttu-id="b78a9-104">På sidan **Regler för betalningskoppling** ställer du in regler som styr hur betalningstext (på en banktransaktion) automatiskt matchas med text på öppna transaktioner i följande två processer:</span><span class="sxs-lookup"><span data-stu-id="b78a9-104">On the **Payment Application Rules** page, you set up rules to govern how payment text (on a bank transaction) is automatically matched with text on open entries in the following two processes:</span></span>
